Module: CustomerNavbar

Provides a navigation bar for customers that includes dynamic links, a cart system, and transaction functionalities. This component integrates with the TransactionContext to manage shopping cart and transaction states. It also handles dynamic UI changes based on screen size and cart interactions.

Parameters:
Name Type Description
links Array.<Object>

An array containing link objects for navigation. Each link object should have 'route' and 'name' properties.
