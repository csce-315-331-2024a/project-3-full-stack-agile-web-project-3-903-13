Module: UpdateOrder/Layout

Provides a layout for the employee section of the application, including a navigation bar and a main content area. This component wraps its children with a TransactionProvider to provide transaction context across all its children. It uses the Suspense component to handle the loading state of its children, providing a better user experience during data fetching. The navigation bar (EmployeeNavbar) includes links to various management sections such as inventory, menu items, and reports.
