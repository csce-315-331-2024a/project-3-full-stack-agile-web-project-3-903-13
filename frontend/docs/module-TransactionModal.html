(static) TransactionModal(props) → {React.Component}

Component that provides a modal to display and manage details of a specific transaction. It allows users to view transaction details such as items, quantity, cost, and status. Users can also cancel in-progress transactions or navigate to update the transaction's details.

(async, static) handleCancel()

Handles the cancellation of the transaction. Attempts to cancel the transaction by sending a PATCH request to the server. If successful, the transaction is removed from the display list or all data is cleared if it's the last transaction.
