Module: ExcessReportPage

A React component that fetches and displays an excess inventory report. The report shows items that were used less than a certain threshold percentage. It allows users to select a date range to generate the report for a specific period.

(async, static) fetchExcessReport()

Initiates the fetching of the excess report based on the selected date range. It handles all aspects of the data fetching process including setting loading states, handling errors, and formatting the received data to be displayed in the UI.
